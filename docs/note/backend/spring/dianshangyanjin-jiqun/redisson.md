# Redisson入门

## 介绍
1. Redisson架设在Redis基础上的一个Java驻内存数据网格（In-Memory Data Grid）
2. 基于NIO的Netty框架，充分利用Redis键值数据库提供的优势
3. 在Java实用工具包中常用接口的基础上，提供一系列具有分布式特性的常用工具类
4. 使原本作为协调单机多线程并发程序的工具包获得了协调分布式多机器多线程并发系统的能力，大大降低了设计和研发大规模分布式系统的难度
